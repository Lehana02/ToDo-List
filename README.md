#  React To-Do List (TaskManager)

A simple and elegant task management application built using **React**, with features like local storage, task editing, and filtering for completed tasks.

---

##  Features

-  Add, edit, and delete todos
-  Mark tasks as completed or incomplete
-  Toggle visibility of completed tasks
-  Persistent storage using **Local Storage**
-  Real-time UI updates
-  Clean and responsive UI (uses **Tailwind CSS**)

---


##  Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Lehana02/ToDo-List.git
cd ToDo-List

```
### 2. Install Dependencies

```bash
npm install

```
### 3. Start the App

```bash
npm run dev

```
